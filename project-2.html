Project Overview

StudyNotion - Edtech platform uses the latest technologies such as React Js and Redux Toolkit, MongoDb as a database, Cloudinary as media player service, Razorpay for payments, and more. StudyNotion is a software application that enables the creation, management, and delivery of educational content and resources to learners. StudyNotion is designed to provide a seamless and efficient learning experience to users. The use of modern technologies such as React Js and Redus Toolkit ensures that the platform is fast, responsive, and scalable. MongoDb as a database and Cloudinary as media player service ensure that the platform can handle large amounts of data and deliver high-quality media content. Razorpay for payments provides a secure and reliable payment gateway for users to purchase courses and other educational resources.
